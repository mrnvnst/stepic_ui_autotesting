## Учебный курс по автоматизации тестирования на Python + Selenium
### Репозиторий предназначен для практики в рамках курса по автоматизации тестирования.

#### Структура

Каждый урок содержит несколько практических задач (task_01, task_02, ...) в формате .md файла и их решение – .py файл.
