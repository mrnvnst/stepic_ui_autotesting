## Учебный курс по автоматизации тестирования на Python + Selenium
### Репозиторий предназначен для практики в рамках курса по автоматизации тестирования.

#### Структура
Материал организован по урокам (lesson_01, lesson_02, ...).
Каждый урок содержит несколько практических задач (task_01, task_02, ...) в формате .md файла и их решение – .py файл.


- `README.md` - описание проекта
- `lesson_01/` – модуль 1: Поиск элементов с помощью Selenium WebDriver
- `lesson_02/` – модуль 2: Полезные методы Selenium
- `lesson_03/` – модуль 3: Тестовые фреймворки
- `requirements.txt` - используемые зависимости

Репозиторий будет развиваться и дополняться по мере прохождения курса.
