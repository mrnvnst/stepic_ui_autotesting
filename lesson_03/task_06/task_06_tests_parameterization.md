### Задание: параметризация тестов

Инопланетяне оставляют загадочные сообщения на Stepik в фидбеке задач на правильное решение. Мы смогли локализовать несколько url-адресов задач, где появляются кусочки сообщений. Ваша задача — реализовать автотест со следующим сценарием действий: 

1. Открыть страницу.
2. Авторизоваться на странице со своим логином и паролем.
3. Ввести правильный ответ.

Правильным ответом на задачу в заданных шагах является число:

```python
import time
import math

answer = math.log(int(time.time()))
```
4. Нажать кнопку "Отправить".
5. Дождаться фидбека о том, что ответ правильный (он указан в подсказке над полем ответа – белый текст на черном фоне).
6. Проверить, что текст в опциональном фидбеке полностью совпадает с "Correct!". Для проверки стоит использовать осмысленное сообщение об ошибки.

Примечания:

- Нужно использовать маркировку pytest для параметризации, передать в тест список ссылок в качестве параметров:

```

https://stepik.org/lesson/236903/step/1
https://stepik.org/lesson/236904/step/1
https://stepik.org/lesson/236905/step/1
```

- Настроить нужные ожидания для стабильной (что маловероятно) работы теста.
-  В упавших тестах найти кусочки послания. Тест должен падать, если текст в опциональном фидбеке не совпадает со строкой "Correct!" 
- Дополнительно убедитсья в том, что на компьютере установлено правильное [локальное время](https://time.is/ru/). Ответ для каждой задачи нужно пересчитывать отдельно, иначе они устаревают.
