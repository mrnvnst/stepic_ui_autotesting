### Задание: поиск элемента по тексту в ссылке

На странице найти зашифрованную ссылку и кликнуть по ней.

1. Открыть страницу: http://suninjuly.github.io/find_link_text.
2. Найти ссылку с текстом. Текст ссылки который нужно найти, зашифрован формулой:
```
 str(math.ceil(math.pow(math.pi, math.e)*10000)) # 224592
```
*Можно вставить данное выражение в свой код, а можно выполнить в интерпретаторе и скопировать оттуда результат. При первом варианте не забыть импортировать math.*

3. Добавить команду для клика по найденной ссылке: она перенесет на форму регистрации.
4. Заполнить форму регистрации.
5. Получить окно с проверочным кодом и вручную скопировать его.
6. Вручную закрыть окно.
7. Закрыть браузер.

Время для заполнения формы ограничено.
